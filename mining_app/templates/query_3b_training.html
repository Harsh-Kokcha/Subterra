Track and monitor worker training completion and certification status

📚 What This Query Shows:

Comprehensive analysis of worker training status including:

  • Total Trainings: All training records for each worker
  • Completed Trainings: Successfully finished courses
  • Expired Trainings: Outdated or expired certifications
  • Active Certifications: Currently valid certificates with expiry dates
  • Certification Status: Classification (Highly Certified, Certified, etc.)
  • Completion Rate: Percentage of trainings completed

SQL Complexity: CASE WHEN, String Aggregation (LISTAGG), Multiple Conditions, GROUP BY

🔍 SQL Query Breakdown

SELECT 
    w.WorkerID, w.Name, w.Role,
    COUNT(DISTINCT tr.TrainingID) AS TotalTrainings,
    COUNT(CASE WHEN tr.TrainingStatus = 'Completed' THEN tr.TrainingID END) AS Completed,
    COUNT(CASE WHEN tr.TrainingStatus = 'Expired' THEN tr.TrainingID END) AS Expired,
    LISTAGG(tr.Course, ', ') WITHIN GROUP (ORDER BY tr.ValidUntil DESC) AS Certifications,
    CASE 
        WHEN COUNT(CASE WHEN tr.TrainingStatus = 'Completed' THEN 1 END) >= 3 THEN 'HIGHLY CERTIFIED'
        ELSE 'CERTIFIED'
    END AS CertificationStatus
FROM Worker w
LEFT JOIN Training_Record tr ON w.WorkerID = tr.WorkerID
GROUP BY w.WorkerID, w.Name, w.Role
ORDER BY Completed DESC;
