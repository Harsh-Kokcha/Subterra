🔍 SQL Query Breakdown

SELECT 
    Location,
    COUNT(*) AS TotalIncidents,                    -- Total incidents per location
    AVG(Severity) AS AvgSeverity,                  -- Average severity
    COUNT(CASE WHEN Severity >= 4 THEN 1 END) AS CriticalIncidents,  -- High severity count
    PERCENT_RANK() OVER (ORDER BY AVG(Severity)) AS SeverityRank     -- Window function ranking
FROM Safety_Incident
GROUP BY Location
ORDER BY AvgSeverity DESC;

Key Concepts:

  • GROUP BY: Aggregates incidents by location
  • CASE WHEN: Counts critical incidents conditionally
  • Window Functions: PERCENT_RANK for relative severity ranking
  • Aggregate Functions: COUNT, AVG, MAX, MIN
